## gh-profile

Work with multiple GitHub accounts using the gh cli

### Options

```
  -c, --config-dir string   GitHub CLI config dir (default "$HOME/.config/gh")
  -h, --help                help for gh-profile
  -v, --version             version for gh-profile
```
